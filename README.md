# FaceVitals

**心率 · HRV · 情绪预测系统**

基于 rPPG（远程光电容积描记）技术的无接触式生理指标监测应用。通过普通摄像头实时检测心率、心率变异性（HRV）和情绪状态。

## 功能特性

| 功能 | 说明 |
|------|------|
| ❤️ 心率监测 | 通过面部皮肤颜色变化提取脉搏信号，实时显示 BPM |
| 💓 HRV 分析 | 计算 RMSSD 指标，评估自主神经系统状态 |
| 🧠 情绪识别 | 结合生理信号和表情特征判断情绪状态 |
| 📹 多摄像头 | 支持电脑摄像头、手机前置/后置摄像头 |
| 📊 信号质量 | 实时显示信号强度，指导用户调整姿势 |

## 技术原理

### rPPG 原理

当心脏跳动时，血液流经面部皮肤，使皮肤颜色发生细微变化。rPPG 技术通过以下步骤提取脉搏信号：

1. **人脸检测** - 使用 MediaPipe 检测面部标志点和区域
2. **ROI 提取** - 提取额头、脸颊等感兴趣区域的像素均值
3. **信号处理** - 使用盲源分离、带通滤波等技术提取纯净脉搏信号
4. **频谱分析** - FFT 变换识别心率峰值

### HRV 分析

心率变异性（Heart Rate Variability）反映自主神经系统的活性。通过分析脉搏信号的间期变化，计算 RMSSD（相邻RR间期差值的均方根）指标。

### 情绪识别

结合多种特征进行情绪判断：
- **生理信号** - 心率水平、HRV 状态
- **表情特征** - 面部动作单元（Action Units）

## 使用指南

### 基础操作

1. **选择摄像头** - 从下拉菜单选择要使用的摄像头
2. **点击开始** - 点击「▶ 开始」按钮启动监测
3. **调整姿势** - 确保面部正对摄像头，光线充足
4. **查看结果** - 心率、HRV 和情绪状态实时显示

### 获取最佳效果

| 要点 | 说明 |
|------|------|
| 💡 光线充足 | 自然光或均匀照明效果最佳，避免直射强光 |
| 😊 面部稳定 | 保持头部相对稳定，避免剧烈晃动 |
| 📏 适当距离 | 面部占据画面 1/3~1/2 区域 |
| 🚫 避免逆光 | 背光会导致信号质量大幅下降 |

### 摄像头配置

程序支持三种摄像头类型：

#### 本地摄像头

直接使用系统摄像头，通过索引选择：
- `0` - 系统默认摄像头
- `1` - 第二个摄像头
- `2` - 第三个摄像头

#### 手机摄像头（网络）

通过 IP 摄像头 App 连接手机摄像头：

1. 在手机上安装 **EpocCam**、**DroidCam** 或类似 App
2. 确保手机和电脑处于同一 WiFi 网络
3. 获取手机分配的 IP 地址和端口号
4. 在应用「⚙ 设置」中添加 URL 类型摄像头

**EpocCam 示例**：
```
http://192.168.1.100:4747/video
```

**DroidCam 示例**：
```
http://192.168.1.100:4747/mjpegfeed
```

#### 默认配置

```python
DEFAULT_CAMERAS = [
    {'name': '💻 电脑摄像头', 'source': '0', 'type': 'local'},
    {'name': '📱 手机后置', 'source': '1', 'type': 'local'},
    {'name': '📱 手机前置', 'source': '2', 'type': 'local'},
]
```

如需修改摄像头顺序，点击「⚙ 设置」进入管理界面编辑。

## 算法参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `buffer_size` | 900 | 信号缓冲大小（约30秒@30fps） |
| `fs` | 30 | 采样率（帧/秒） |
| `min_signal_len` | 100 | 计算 HRV 所需最小帧数 |

如需调整算法灵敏度，可修改 `core/` 目录下对应模块的参数。

## 常见问题

### 无法打开摄像头

1. 检查系统隐私设置，确保应用有摄像头权限
2. 确认摄像头未被其他程序占用
3. 点击「🔄 切换」按钮刷新摄像头列表
4. 手动在设置中输入正确的摄像头索引

### 心率显示"--"或数值不稳定

1. 确保光线充足且均匀，避免强光直射
2. 保持面部正对摄像头，表情放松
3. 等待 5-10 秒信号稳定后再查看结果
4. 避免在运动后立即测量

### 如何提高测量准确性

1. 使用外置高清摄像头效果更好
2. 环境光线以柔和均匀为佳
3. 测量时保持安静，避免说话
4. 等待足够长的信号采集时间

### 手机摄像头连接失败

1. 确认手机和电脑在同一网络
2. 检查 IP 地址和端口是否正确
3. 确保手机端 App 已启动并正常推流
4. 检查防火墙是否阻止了连接
